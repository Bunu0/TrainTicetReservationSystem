# 🚆 Train Ticket Reservation System

## 📌 Overview
The Train Ticket Reservation System is a Java-based application designed to simplify the process of booking, managing, and canceling train tickets. It provides a structured and efficient way to handle passenger details, train schedules, and reservations.

This project follows a modular approach using core Java concepts and database integration.

---

## 🚀 Features

- 🔍 Search trains by source and destination  
- 🎫 Book train tickets  
- ❌ Cancel reservations  
- 👤 Manage passenger details  
- 💺 Check seat availability  
- 📊 Maintain booking records  

Such systems are commonly used to automate ticket booking and improve user experience. :contentReference[oaicite:1]{index=1}

## ⚙️ Installation & Setup

### 1. Clone the Repository

git clone https://github.com/Bunu0/TrainTicetReservationSystem.git


### 2. Navigate to Project Folder

cd TrainTicetReservationSystem


### 3. Configure Database
- Create a MySQL database  
- Update credentials in your code / `application.properties`

### 4. Run the Application
- Run main class in your IDE  
OR  

mvn spring-boot:run
